# Product

## Register

product

## Users

ToMaple is for people who manage personal focus, daily schedules, project goals, notes, and preparation routines in one compact workspace. They use it during planning and work sessions, so the interface should stay calm, direct, and easy to scan.

## Product Purpose

The product helps users move from planning to focused execution: schedule work blocks, run focus sessions, track projects and goals, collect notes, and prepare recurring reminders. Success means every visible control either performs a useful action or clearly explains its current state.

## Brand Personality

Calm, capable, organized. The voice should feel supportive and practical, with Traditional Chinese labels that describe the action plainly.

## Anti-references

Avoid marketing-page theatrics, decorative UI that competes with task work, hidden controls that appear broken, and settings that only look clickable. Avoid overusing saturated color on inactive controls.

## Design Principles

- Make every control accountable: clickable elements should navigate, open a menu, change state, or provide feedback.
- Keep the workspace quiet: use restrained color and predictable product UI patterns.
- Preserve continuity: local data and preferences should remain in the browser and survive refreshes.
- Prefer clear empty states: empty lists should tell the user what they can do next.
- Make theme choice explicit: light and dark mode should be controlled from the top-right settings menu.

## Accessibility & Inclusion

Target practical WCAG AA contrast for text and controls. Preserve keyboard focus affordances, keep motion short and purposeful, and respect reduced-motion preferences for decorative transitions.
